## Crisis Management

### If things go wrong...

**Site crashes:**
1. Check Vercel status
2. Review error logs (Sentry)
3. Post status update on X
4. Fix critical issues first
5. Communicate ETA to users

**OpenAI rate limits:**
1. Implement queue system
2. Show estimated wait time
3. Upgrade OpenAI tier
4. Add Redis caching

**Payment issues:**
1. Check Stripe dashboard
2. Verify webhook endpoint
3. Test in incognito mode
4. Contact Stripe support if needed

**Bad press/reviews:**
1. Respond professionally and quickly
2. Acknowledge legitimate issues
3. Share how you're fixing them
4. Turn critics into fans
